# Snow Mobile Core
## Suitcase-sized device which provides a suite of services through a 5G or 4G connection.
**Ownership**: Amazon
**Type**: IaaS
**Services Provided**: 
- Config
- CloudWatch
- AutoScaling
- CloudFormation
- CloudTrail
- S3
- SAS
- Systems Manager
- Provisioning and Subscriber tools
- VPC Management Tools
- Mobile core control plane.
